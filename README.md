# 这是一个VUE项目

## 一、使用
### 1、npm i (安装依赖)
### 2、npm run dev （开发运行）
### 3、webpack （打包-dist目录下）

## 二、API或许已经失效

### 1、本项目使用假数据 ，若要用真数据 替换有效的api接口即可 ， 并在组件的template中替换接收真数据的变量名
### 2、假数据中的图片使用地址为 ./src/image(开发阶段)  打包后在dist文件下打开index.html文件图片地址会失效 更换为../src/images即可
### 3、真数据中的图片是在服务器上，不存在此问题
